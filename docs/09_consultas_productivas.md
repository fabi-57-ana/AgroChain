# Consultas productivas

## Objetivo

Realizar consultas SQL sobre la información histórica de producción de maní
almacenada en AgroChain, generando datos preparados para futuras etapas de
trazabilidad mediante blockchain.


## Consultas realizadas


### 1. Evolución histórica

Se consultó la evolución anual de producción y rendimiento desde 1927 hasta 2024.

Resultado:

- Registros analizados: 98 años.
- Periodo: 1927-2024.
- Se observa un cambio productivo significativo a partir del año 2000,
  asociado a incorporación tecnológica y mejoras productivas.


### 2. Producción por provincia - campaña 2024

Se obtuvo la distribución territorial de la producción.

Principales provincias:

- Córdoba: 1.268.363 tn
- Buenos Aires: 315.761 tn
- La Pampa: 127.986 tn


### 3. Ranking departamental

Se identificaron los principales departamentos productores.

Primeros resultados:

- Río Cuarto (Córdoba): 297.500 tn
- General Roca (Córdoba): 249.150 tn
- Juárez Celman (Córdoba): 180.084 tn


### 4. Periodo tecnológico

Se analizó el periodo 2000-2024 para observar la evolución reciente.

Se confirma un incremento importante de rendimiento y producción respecto
al periodo histórico anterior.


### 5. Preparación para blockchain

Se generó una consulta estructurada con información productiva territorial.

Los datos obtenidos podrán utilizarse como eventos de trazabilidad,
registrando:

- cultivo
- año
- provincia
- departamento
- producción
- rendimiento
